# Sparlos Technical Training Center

A modern and responsive website for **Sparlos**, a **Technical and Vocational Training Institute**, built with **HTML5**, **CSS3**, and **jQuery**. This project showcases course listings, instructor profiles, and student registration options in a clean and accessible layout.

## 🏫 Overview
This is a multi-page static website designed for the **Sparlos Technical Training Center**, offering users the ability to:
- Browse available courses and instructors
- View detailed course information
- Explore teacher profiles
- Learn about the institute
- Navigate through a fully responsive interface

All assets are organized into clear directories for easy customization and maintenance.

## 📋 Pages Included
- `index.html` – Main homepage
- `teachers.html` – Instructor listing
- `gallery.html` – Media gallery
- `courses.html` – Course listing
- `course-info.html` – Detailed view of a specific course
- `about-us.html` – About the institute

## 💻 Technologies Used
- **HTML5** – Semantic markup and accessibility
- **CSS3** – Responsive layout, animations, Flexbox/Grid
- **jQuery** – Interactive elements like sliders, tabs, and modal popups

> ⚠️ Note: This is a frontend-only project. It can be extended with backend integration for real course registration.
